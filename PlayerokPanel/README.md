# Калькулятор маржинальности Playerok / FunPay

Удобный веб-калькулятор и менеджер товаров для продавцов на площадках Playerok и FunPay. Приложение работает полностью в браузере и не требует никаких переменных окружения (`.env`) или баз данных — все ваши данные надёжно сохраняются локально на вашем устройстве (в браузере через `localStorage`).

---

## Как скачать ZIP-архив с кодом из AI Studio

1. Посмотрите в верхний правый угол интерфейса **Google AI Studio**.
2. Нажмите на иконку настроек (шестерёнку) или кнопку экспорта/меню.
3. Выберите опцию **Export** (Экспорт) -> **Download ZIP** (Скачать ZIP-архив).
4. Сохраните скачанный архив на свой компьютер и распакуйте его в любую удобную папку.

---

## Как запустить приложение на своём компьютере

Для работы приложения вам понадобится установленный **Node.js** (рекомендуется версия 18 или выше).

### Шаг 1: Установка зависимостей
Откройте терминал/командную строку в папке с проектом и выполните команду:
```bash
npm install
```

### Шаг 2: Запуск в режиме разработки
Для запуска локального сервера выполните:
```bash
npm run dev
```
После запуска консоль покажет ссылку (обычно `http://localhost:3000` или `http://localhost:5173`). Откройте её в любимом браузере!

### Шаг 3: Сборка для публикации (Production)
Если вы хотите собрать готовый оптимизированный сайт для размещения на хостинге:
1. Соберите проект:
   ```bash
   npm run build
   ```
2. Посмотрите собранную версию локально:
   ```bash
   npm run preview
   ```
   Готовые файлы будут находиться в папке `dist/`, их можно загрузить на любой бесплатный хостинг (например, GitHub Pages, Vercel, Netlify).

---

## Особенности сборки без .env
* **Нулевая конфигурация**: Для запуска проекта **не требуются** никакие `.env` ключи или токены.
* **Автономность**: Всё рассчитывается напрямую прямо на веб-странице, поэтому проект готов к работе сразу после распаковки.
